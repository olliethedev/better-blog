### Prerequisites

- You must have [shadcn/ui](https://ui.shadcn.com) installed with CSS variables enabled.
- Configure a [Sonner](https://ui.shadcn.com/docs/components/sonner) `<Toaster />` component for toast notifications.
- Ensure you have [TailwindCSS](https://tailwindcss.com) `v4` set up and configured correctly.

### Install the package

```bash
pnpm add better-blog
# or
npm install better-blog
# or
yarn add better-blog
```

### Tailwind CSS content configuration

Better Blog ships utility classes. Configure Tailwind to scan the package.

- [TailwindCSS](https://tailwindcss.com) v4 (in your `globals.css`):

```css
/* globals.css */
@import "better-blog/css";
```

- For [TailwindCSS](https://tailwindcss.com) V3 (Deprecated), add the following `content` to your Tailwind config:

```ts
// tailwind.config.ts
export default {
  content: [
    // your app paths...
    "./node_modules/better-blog/dist/**/*.{js,ts,jsx,tsx,mdx}",
  ],
};
```
